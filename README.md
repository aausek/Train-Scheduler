# Train-Scheduler

### The Train Scheduler allows the user to enter new train information appending train name, destination, frequency, arrival and time remaining.
### This FireBase App utilizes the Google Tool to store data remotely
### The app also incorporates Moment.js to determine train arrival and departure times
